Final-version Instructions for XYZ 2013

The proceedings of XYZ 2013 will be published in Springer′s Lecture Notes in Computer Science (LNCS). Authors must prepare their camera-ready version in LaTeX2e. You should obtain the LNCS LaTeX2e class file (llncs.cls), which can be obtained off Springer's site at:

http://www.springer.de/comp/lncs/authors.html

Click on the link "Information for LNCS Authors", which will take you to a page with a zip file that includes the class file and documentation on how to use that class file. You should ignore the information about the Springer's copyright form and you must not sign Springer's LNCS copyright form.

The page limit for your camera-ready paper is 18 pages total, including bibliography and all appendices. Please prepare an archive file (either zip, tar, tar.gz, or tgz) that includes the following files:

  • The source LaTeX file, including any figures, style files, and all other files that are needed to produce the camera ready copy.
  • A PDF file that was created from the source LaTeX file(s).
Then access the web-page at [[[...]]]/submit/cameraready.php to upload it. On the camera-ready page you also need to upload the PDF file separately. (This means that the PDF file is uploaded twice, once by itself and once inside the archive file.) Before you can upload your camera-ready version, you must sign electronically the IACR copyright form.

The deadline for uploading the files is [[[camera-ready deadline]]], this is a firm deadline.

ePrint. If you have an ePrint report for this work, you should indicate the report number on the camera-ready form. If you still do not have an ePrint report for it, you are strongly encouraged to create one, otherwise one will be created for you sometime before the conference. The camera-ready version that you upload to the server will be available off of this ePrint report, in addition to any version that you already have there (if any). See the IACR copyright form for more information.

NOTE: you will need the submission-ID and password that were given to you when you first submitted the paper. Contact the chair if you lost your submission-ID and password.
